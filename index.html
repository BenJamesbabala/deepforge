DeepForge

Development Environment for Deep Learning with Torch

  • Learn more

Build Architectures

Use DeepForge's visual architecture editor to create and edit neural network architectures. Easily drop down to a torch text editor to define custom torch layers and immediately use them in your architectures!

Create Training Pipelines

Compose training and testing pipelines from torch operations. Supports easily dropping into a torch text editor to define custom operation.

Execute Pipelines

After creating your pipelines, execute the operations over a number of distributed worker machines and view the progress and results of each remote operation from within the browser!

Installation

DeepForge can be easily installed on OSX (and some linux distros) with the following command:

curl -o- https://raw.githubusercontent.com/dfst/deepforge/master/install.sh | bash


To install DeepForge for use with distributed pipeline execution, check out the custom installation options!
